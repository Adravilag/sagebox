# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2025-12-08

### Added
- Initial release of SageBox UI component library
- Core web components built with Stencil.js
- Design tokens system with CSS custom properties
- Angular standalone wrappers
- React wrappers
- Icon system with CLI tool
- Full test coverage infrastructure
- CI/CD pipeline with GitHub Actions
- Documentation site

### Components
- `sg-article-editor` - Rich text editor
- `sg-badge` - Status badges
- `sg-breadcrumb` - Navigation breadcrumbs
- `sg-button` - Buttons with variants
- `sg-card` - Content cards
- `sg-context-menu` - Right-click menus
- `sg-date-picker` - Date selection
- `sg-dropdown` - Dropdown menus
- `sg-form-section` - Form grouping
- `sg-icon` - SVG icons
- `sg-info-field` - Information display
- `sg-input` - Text inputs
- `sg-modal` - Modal dialogs
- `sg-search-box` - Search input
- `sg-select` - Select dropdowns
- `sg-skeleton` - Loading skeletons
- `sg-stats-card` - Statistics cards
- `sg-theme-toggle` - Dark/light mode toggle
- `sg-tooltip` - Tooltips
